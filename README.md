# Product Management REST API

This project is a Spring Boot based RESTful API that performs CRUD operations on a Product resource.  
It follows layered architecture (Controller, Service, Repository, DTO), includes validation, exception handling, and Swagger documentation.

---

##  Tech Stack
- Java 8
- Spring Boot 2.8.x
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- Swagger (OpenAPI)
- Lombok
---

##  Features
- Create, Read, Update, Delete Product
- DTO-based request and response handling
- Bean Validation (JSR-380)
- Global Exception Handling
- Swagger UI for API documentation
- Clean layered architecture
